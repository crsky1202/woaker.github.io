题目

方法

双指针，其实就是一个指针固定第一个元素上，第二个指针遍历数组。

  • 比如，3，2，2，3。 用一个变量p和i在3元素上的索引，
  • 将索引i的元素和val的值进行比较
  • 如果值不相同，将索引i的元素转移到索引p的位置上，并且将p++
  • 如果相同，就i依然跟着数组遍历
